# 🌱 Agro Forte, Futuro Sustentável

Projeto desenvolvido para o concurso Agrinho 2026.

## 📌 Objetivo

Demonstrar a importância do equilíbrio entre produção agrícola e preservação ambiental através de um site interativo utilizando HTML, CSS e JavaScript.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## 📱 Funcionalidades

- Modo escuro
- Responsividade
- Contador ambiental
- Formulário interativo
- Animações suaves
- Navegação intuitiva
